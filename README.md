# 旧项目归档:工厂门禁人员IPAD/手机查询系统

# 架构
基于dedeCMS构建,PHP5.2 MYSQL5.5,生命周期2013年11月至今

# 功能说明

集团门禁人员查询识别系统，对厂区进出人员实现方便、快捷的查询识别功能。门卫人员可以根据进出人员提供的姓名、职工编号、工作单位等任意一种人员特征，使用系统快速查询得到人员的基本信息。
